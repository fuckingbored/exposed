# Usage
<!-- usage -->
```sh-session
$ npm install -g exposed
$ xps COMMAND
running command...
$ xps (-v|--version|version)
exposed/0.0.0 win32-x64 node-v12.13.0
$ xps --help [COMMAND]
USAGE
  $ xps COMMAND
...
```
